Radio Control 1.2.9

Автор : Radius

Описание :

Плагин предоставляет широкие возможности по управлению работой и настройкой большинства радиосообщений, включая меню радиосообщений. Все меню и сообщения уже переведены на русский язык, также имеется возможность перевести на необходимое количество языков.

Особенности :

  • работает с последними билдами 7561 (Windows) и 7559 (Linux)
  • возможность полного вырезания чат сообщения, звука и иконки над головой при броске Осколочной, Свето-Шумовой и Дымовой гранат (квары rc_remove_he, rc_remove_flash, rc_remove_smoke)
  • возможность задать временной промежуток (в секундах) между двумя радиосообщениями для борьбы с флудом радиокомандами (квар rc_radio_time)
  • возможность задать количество сообщений которые можно отправить в течении одного раунда (квар rc_radio_msgs)
  • новые меню радиосообщений, с возможностью изменения по своему вкусу и переводом на любое количество языков (квар rc_new_radio_menus)
  • новые цветные чат сообщения для радиокоманд, с возможностью изменения по своему вкусу и переводом на любое количество языков (квар rc_new_radio_chat_msgs)
  • возможность полностью отключить текст радиосообщений (квар rc_send_chat_msg)
  • возможность полностью отключить звуки радиосообщений (квар rc_play_sound)
  • все квары и команды для удобства настраиваются в автоматически создаваемом конфигурационном файле 'addons/amxmodx/configs/radio_control.cfg'
  • автоматически создаваемый словарь многоязычности 'addons/amxmodx/data/lang/radio_control.txt'

Требования:

  • Plugins Core

Настройки (из автоматически создаваемого конфигурационного файла 'radio_control.cfg'):

// Уровень дебаг-логов. 0 - отключено // По умолчанию: "0" rc_debug_level "0" // Отключает (1) радиосообщение при броске осколочной гранаты (чат сообщение, звук, иконка над головой) // По умолчанию: "0" rc_remove_he "0" // Отключает (1) радиосообщение при броске дымовой гранаты (чат сообщение, звук, иконка над головой) // По умолчанию: "0" rc_remove_smoke "0" // Отключает (1) радиосообщение при броске свето-шумовой гранаты (чат сообщение, звук, иконка над головой) // По умолчанию: "0" rc_remove_flash "0" // Временной промежуток (в секундах) между двумя радиосообщениями (поможет устранить флуд) // По умолчанию: "1.5" rc_radio_time "1.5" // Количество сообщений которые можно отправить в течении одного раунда (счетчик обнуляется при возрождении игрока) // По умолчанию: "60" rc_radio_msgs "60" // Использовать (1) или нет (0) нестандартные меню радиосообщений // По умолчанию: "1" rc_new_radio_menus "1" // Использовать (1) или нет (0) нестандартные чат сообщения для радиокоманд // По умолчанию: "1" rc_new_radio_chat_msgs "1" // Отправлять (1) или нет (0) в чат текстовое сообщение радиокоманды // По умолчанию: "1" rc_send_chat_msg "1" // Проигрывать (1) или нет (0) звук радиосообщения // По умолчанию: "1" rc_play_sound "1" // Показывать (1) или нет (0) иконку восклицательного знака над головой во время отправки радиосообщения // По умолчанию: "1" rc_show_icon "1" // Добавлять (1) или нет (0) цветное описание типа гранат в радиосообщении Fire in the hole! // По умолчанию: "1" rc_fire_in_the_hole_advanced "1"
