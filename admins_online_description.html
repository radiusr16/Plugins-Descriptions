Admins Online 1.0.0

Автор: Radius

Описание:

Плагин информирует игроков о наличии администраторов на сервере. Обладает широкими возможностями настройки.

Особенности:

  • возможность вывода списка админов в виде чат, HUD или DHUD сообщений (квар ao_msg_type)
  • возможность вывода списка админов с именами или только их количества (квар ao_full_admins_list)
  • возможность отключения и включения отображения списка админов с сохранением выбора с помощью команд чата: /admin, !admin, /admins, !admins
  • возможность задать событие при котором будет выводиться список админов: в начале каждого раунда, после смерти игрока, периодически пока игрок мертв, периодически (квар ao_show_event)
  • возможность настроить флаги доступа, по которым будет формироваться список админов (квар ao_admin_flags)
  • возможность настроить RGB цвет, XY координаты и продолжительность HUD/DHUD сообщений (квар ao_hud_msg)
  • все квары и команды для удобства настраиваются в автоматически создаваемом конфигурационном файле 'addons/amxmodx/configs/admins_online.cfg'
  • автоматически создаваемый словарь многоязычности 'addons/amxmodx/data/lang/admins_online.txt'

Требования:

  • Plugins Core (версии 1.8.0 и выше)

Настройки (из автоматически создаваемого конфигурационного файла 'admins_online.cfg'):

// Уровень дебаг-логов. 0 - отключено // По умолчанию: "0" ao_debug_level "0" // Задает тип сообщения. 0 - чат, 1 - HUD, 2 - DHUD // По умолчанию: "0" ao_msg_type "0" // Формировать полный список админов (1) или только их количество (2) // По умолчанию: "1" ao_full_admins_list "1" // Событие, при котором будет автоматически отображаться список админов: 0 - отключено, 1 - в начале каждого раунда, 2 - после смерти игрока, 3 - периодически пока игрок мертв, 4 - периодически // По умолчанию: "1" ao_show_event "1" // Флаги доступа для админов // По умолчанию: "d" ao_admin_flags "d" // RGB цвет, XY координаты и продолжительность HUD/DHUD сообщения со списком админов в игре // По умолчанию: "220 160 0 | 0.02, 0.26 | 10.0" ao_hud_msg "220 160 0 | 0.02, 0.26 | 10.0"
