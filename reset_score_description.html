Reset Score 1.5.0

Автор: Radius

Благодарности: Bullseye

Описание:

Плагин позволяет обнулять счет с помощью команд, автоматически или через удобное меню, обладает широкими возможностями настройки.

Особенности:

  • возможность обнуление счета с помощью настраиваемых команд (квар rs_reset_score_cmd)
  • возможность обнуление счета за деньги (квар rs_score_reset_cost)
  • возможность сохранения и загрузки счета игрока при выходе с сервера (квар rs_save_score)
  • возможность настройки доступа к обнулению счета (квары rs_access_flags и rs_high_access_flags)
  • звуковое сопровождение обнуления счета (квары rs_reset_score_sound и rs_not_resetted_sound)
  • сохранение личных настроек и восстановление при следующем подключении
  • имеется API для разработчиков (см 'addons/amxmodx/scripting/reset_score.inc')
  • все квары и команды для удобства настраиваются в автоматически создаваемом конфигурационном файле 'addons/amxmodx/configs/reset_score.cfg'
  • автоматически создаваемый словарь многоязычности 'addons/amxmodx/data/lang/reset_score.txt'

Требования:

  • Plugins Core (версии 1.8.0 и выше)

Настройки (из автоматически создаваемого конфигурационного файла 'reset_score.cfg'):
