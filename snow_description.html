Snow 1.1.0

Автор: Radius | floatman (idea)

Описание:

Плагин добавляет стандартный эффект снега и позволяет каждому игроку настроить его интенсивность или отключить с помощью меню и с сохранением выбора.

Особенности:

  • возможность настроить значение интенсивности по умолчанию (квар snow_default_intensity)
  • возможность настройки задержки появления меню после подключения игрока (квар snow_menu_delay)
  • возможность настроить звук, который будет проигрываться игроку при первом открытии меню (квар snow_menu_sound)
  • все квары и команды для удобства настраиваются в автоматически создаваемом конфигурационном файле 'addons/amxmodx/configs/snow.cfg'
  • автоматически создаваемый словарь многоязычности 'addons/amxmodx/data/lang/snow.txt'

Требования:

  • Plugins Core (версии 1.8.0 и выше)

Настройки (из автоматически создаваемого конфигурационного файла 'snow.cfg'):

// Уровень дебаг-логов. 0 - отключено // По умолчанию: "0" snow_debug_level "0" // Значение интенсивности снега по умолчанию // По умолчанию: "1" snow_default_intensity "1" // Доступно (1) или нет (0) меню выбора интенсивности снега // По умолчанию: "1" snow_menu_enabled "1" // Список команд (разделенных ';'), с помощью которых будет вызываться меню выбора интенсивности снега // По умолчанию: "say /snow" snow_menu_cmd "say /snow" // Задержка (в секундах) показа меню выбора интенсивности снега после возврождения (только если не сохранен выбор) // По умолчанию: "5" snow_menu_delay "5" // Путь к файлу звука (mp3 или wav), который будет проигрываться игроку при первом открытии меню // Если квар пустой, то звук не проигрывается // По умолчанию: "buttons/bell1.wav" snow_menu_sound "buttons/bell1.wav" // Время (в днях) хранения данных игрока // По умолчанию: "30" snow_data_storage_time "30"
