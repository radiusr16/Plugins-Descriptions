Snow 1.1.0

Автор: Radius | floatman (idea)

Описание:

Плагин добавляет стандартный эффект снега и позволяет каждому игроку настроить его интенсивность или отключить с помощью меню и с сохранением выбора.

Особенности:

  • возможность через меню настроить значение интенсивности с сохранением выбора и восстановлением при следующем подключении игрока
  • возможность настроить значение интенсивности по умолчанию (квар snow_default_intensity)
  • возможность настройки задержки появления меню после подключения игрока (квар snow_menu_delay)
  • возможность настроить звук, который будет проигрываться игроку при первом открытии меню (квар snow_menu_sound)
  • имеется API для разработчиков
  • все квары и команды для удобства настраиваются в автоматически создаваемом конфигурационном файле 'addons/amxmodx/configs/snow.cfg'
  • автоматически создаваемый словарь многоязычности 'addons/amxmodx/data/lang/snow.txt'

Требования:

  • Plugins Core (версии 1.8.0 и выше)

Настройки (из автоматически создаваемого конфигурационного файла 'snow.cfg'):
