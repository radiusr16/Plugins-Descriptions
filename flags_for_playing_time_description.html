Flags For Playing Time 0.1.1

Автор: Radius

Описание:

Плагин выдает флаги доступа за время проведенное в игре.

Особенности:

  • возможность задать флаги доступа и требуемое время игры
  • возможность задать на какое время выдаются флаги доступа
  • возможность игнорировать игроков с определенными флагами доступа
  • автоматическое или ручное информирование о времени оставшемся до получения или окончания флагов доступа
  • возможность отключить (с сохранением выбора) информер о времени оставшемся до получения или окончания флагов доступа
  • все квары и команды для удобства настраиваются в автоматически создаваемом конфигурационном файле 'addons/amxmodx/configs/flags_for_playing_time.cfg'
  • автоматически создаваемый словарь многоязычности 'addons/amxmodx/data/lang/flags_for_playing_time.txt'

Требования:

  • Plugins Core (версии 1.3.0 и выше)
  • Admin Loader by 'neygomon | Radius Edition' или GameCMS_API by 'zhorzh78'

Настройки (из автоматически создаваемого конфигурационного файла 'flags_for_playing_time.cfg'):

// fpt_update_config - обновить (1) или нет (0) данный файл при смене карты // Внимание: Текущие значения кваров (кроме команд), запишутся в обновленный файл // Квар со значением 1 прописывается в консоли сервера // После обновления квар сам вернется в значение 0 // fpt_update_dictionary - обновить (1) или нет (0) lang файл при смене карты // Внимание: Квар со значением 1 прописывается в консоли сервера // После обновления квар сам вернется в значение 0 // Уровень логов // 0 - выключено (OFF) // 1 - предупреждения (WARN) // 2 - информация (INFO) // 3 - отладочная информация (DEBUG) // 4 - подробная отладочная информация (TRACE) // По умолчанию: "2" fpt_log_level "2" // Необходимое для получения флагов доступа время игры (в минутах) // По умолчанию: "240" fpt_required_play_time "240" // Время, на которое выдаются флаги доступа (в минутах) // По умолчанию: "1440" fpt_flags_duration "1440" // Время, на которое блокируется повторная выдача флагов доступа (в минутах) // Внимание: Время отсчитывается сразу после выдачи флагов доступа // По умолчанию: "1440" fpt_ignore_duration "1440" // Выдаваемые флаги доступа // По умолчанию: "t" fpt_add_flags "t" // Флаги доступа
