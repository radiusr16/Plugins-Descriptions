Flags For Playing Time 0.2.0

Автор: Radius

Описание:

Плагин выдает флаги доступа за время проведенное в игре.

Особенности:

  • возможность задать флаги доступа и требуемое время игры
  • возможность задать на какое время выдаются флаги доступа
  • возможность игнорировать игроков с определенными флагами доступа
  • автоматическое или ручное информирование о времени оставшемся до получения или окончания флагов доступа
  • возможность отключить (с сохранением выбора) информер о времени оставшемся до получения или окончания флагов доступа
  • все квары и команды для удобства настраиваются в автоматически создаваемом конфигурационном файле 'addons/amxmodx/configs/flags_for_playing_time.cfg'
  • автоматически создаваемый словарь многоязычности 'addons/amxmodx/data/lang/flags_for_playing_time.txt'

Требования:

  • Plugins Core (версии 1.8.0 и выше)

Настройки (из автоматически создаваемого конфигурационного файла 'flags_for_playing_time.cfg'):

// fpt_update_config - обновляет файл 'flags_for_playing_time.cfg' при смене карты // 1 - обновить файл // 2 - сделать бэкап (*.bkp) и обновить файл // Внимание: Текущие значения кваров (кроме команд), запишутся в обновленный файл // Квар со значением прописывается в консоли сервера // После обновления квар сам вернется в значение 0 // fpt_update_dictionary - обновляет lang файл 'flags_for_playing_time.txt' при смене карты // 1 - обновить файл // 2 - сделать бэкап (*.bkp) и обновить файл // Внимание: Квар со значением прописывается в консоли сервера // После обновления квар сам вернется в значение 0 // Уровень логов // 0 - выключено (OFF) // 1 - предупреждения (WARN) // 2 - информация (INFO) // 3 - отладочная информация (DEBUG) // 4 - подробная отладочная информация (TRACE) // По умолчанию: "2" fpt_log_level "2" // Куда будут писаться логи: // 0 - в стандартный лог AMX Mod X 'addons/amxmodx/logs', // 1 - в подпапку 'addons/amxmodx/logs/flags_for_playing_time' // По умолчанию: "1" fpt_log_type "1" // Необходимое для получения флагов доступа время игры (в минутах) // По умолчанию: "240" fpt_required_play_time "240" // Время, на которое выдаются флаги доступа (в минутах) // По умолчанию: "1440" fpt_flags_duration "1440" // Время, на которое блокируется повторная выдача флагов доступа (в минутах) // Внимание: Время отсчитывается сразу после выдачи флагов доступа // По умолчанию: "1440" fpt_ignore_duration "1440" // Выдаваемые флаги доступа // По умолчанию: "t" fpt_add_flags "t" // Флаги доступа игроков, которым НЕ БУДЕТ выдаваться бонусные флаги доступа // Если флаги перечислены через запятую, то для игнорирования необходим любой // из них, иначе необходимы все // По умолчанию: "d" fpt_ignore_flags "d" // Список команд (разделенных ';'), с помощью которых можно узнать время // оставшееся до получения или окончания флагов доступа // По умолчанию: "say /ft" fpt_time_info_cmd "say /ft" // Список команд (разделенных ';'), с помощью которых можно включить/отключить // отображение HUD сообщения, информирующего игрока о том когда будут выданы // или кончатся флаги доступа // По умолчанию: "say /fthud" fpt_time_info_hud_cmd "say /fthud" // RGB цвет, XY координаты и продолжительность HUD сообщения, информирующего // игрока о том когда будут выданы или кончатся флаги доступа // По умолчанию: "220 160 0 | 0.02 0.9 | 6.0" fpt_time_info_hud_msg "220 160 0 | 0.02 0.9 | 6.0" // Частота отправки HUD сообщения (в секундах), информирующего игрока о том когда // будут выданы или кончатся флаги доступа // По умолчанию: "120" fpt_time_info_hud_msg_freq "120" // Время хранения данных игрока (в минутах), отвечающих за время игры на сервере // По умолчанию: "5" fpt_data1_storage_time "5" // Время хранения данных игрока (в днях), отвечающих за статус отображения HUD // сообщения // По умолчанию: "30" fpt_data2_storage_time "30"
