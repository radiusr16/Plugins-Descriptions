Slay Losers 1.3.0

Автор: Radius

Описание:

Плагин в конце раунда наказывает выживших игроков проигравшей команды за невыполнение задания карты. Является улучшенной версией оригинального плагина AMX Slay Losers от d3n14@yahoo.com.

Особенности:

  • возможность настройки минимального количества игроков необходимого для работы плагина (квар sl_min_players)
  • возможность настройки иммунитета (квар sl_immunity_flags)
  • возможность включения/отключения эффектов (квар sl_effects)
  • возможность включения/отключения наказания если была попытка разминирования или вывода части заложников (квар sl_ignore_if_tried)
  • возможность настроить тип сообщения: чат, HUD, DHUD (не нужные закомментировать в 'addons/amxmodx/data/lang/slay_losers.txt' при выключенном сервере)
  • возможность задать звук wav или mp3, который будет проигрываться при наказании за невыполнение задания карты (квар sl_announce_sound)
  • все квары и команды для удобства настраиваются в автоматически создаваемом конфигурационном файле 'addons/amxmodx/configs/slay_losers.cfg'
  • автоматически создаваемый словарь многоязычности 'addons/amxmodx/data/lang/slay_losers.txt'

Требования:

  • Plugins Core
  • ReAPI

Настройки (из автоматически создаваемого конфигурационного файла 'slay_losers.cfg'):

// sl_update_config - обновляет файл 'slay_losers.cfg' при смене карты // 1 - обновить файл // 2 - сделать бэкап (*.bkp) и обновить файл // Внимание: Текущие значения кваров (кроме команд), запишутся в обновленный файл // Квар со значением прописывается в консоли сервера // После обновления квар сам вернется в значение 0 // sl_update_dictionary - эффект молнии // b - звук грома // c - эффект крови // По умолчанию: "abc" sl_effects "abc" // Игнорировать (1) или нет (0) наказание команды, если бомба уже была в процессе // разминирования или если часть заложников спасена // По умолчанию: "1" sl_ignore_if_tried "1" // RGB цвет, XY координаты и продолжительность HUD сообщения, информирующего // о том что игроки команды наказаны за невыполнение задания карты // По умолчанию: "139 139 0 | -1.0 0.36" sl_hud_msg "139 139 0 | -1.0 0.36" // Путь к файлу звука (mp3 или wav), который проигрывается при наказании // за невыполнение задания карты // Если значение пустое, то звук не проигрывается // По умолчанию: "slay_losers/slay_losers.wav" sl_announce_sound "slay_losers/slay_losers.wav" // Путь к файлу звука грома (только wav) // Если значение пустое, то звук не проигрывается // По умолчанию: "ambience/thunder_clap.wav" sl_thunder_sound "ambience/thunder_clap.wav"
