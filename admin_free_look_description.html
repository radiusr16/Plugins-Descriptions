Admin Free Look 1.3.5

Автор: Radius

Описание:

Плагин позволяет админам наблюдать за всеми игроками, несмотря на ограничения связанные с кварами mp_forcecamera и mp_forcechasecam.

Также плагин добавляет Free Look меню, в котором админы могут быстро переключиться на нужного игрока или камеру на карте. Меню доступно и для обычных игроков, но список наблюдения ограничен настройками.

Особенности:

  • автоматическое восстановление наблюдения за умершим игроком после возрождения (квар af_lock_target)
  • плагин поддерживает ReApi и последние HLDS билды
  • возможность быстрого переключения по камерам (которые переключаются во время выбора команды) через Free Look меню (квар af_use_trigger_camera)
  • Free Look меню, которое позволяет быстро переключаться на нужного игрока (учитывается квар af_forcecamera)
  • возможность настроить флаги доступа для игроков, которые смогут наблюдать за всеми, несмотря на ограничения связанные с кварами mp_forcecamera и mp_forcechasecam (квар af_access_flags "d")
  • полная свобода выбора режимов наблюдения для игроков с доступом
  • автоматическое выключение плагина если квар af_forcecamera установлен в значение 0
  • все квары и команды для удобства настраиваются в автоматически создаваемом конфигурационном файле 'addons/amxmodx/configs/admin_free_look.cfg'
  • автоматически создаваемый словарь многоязычности 'addons/amxmodx/data/lang/admin_free_look.txt'

Требования:

  • Plugins Core

Настройки (из автоматически создаваемого конфигурационного файла 'admin_free_look.cfg'):

// Уровень дебаг-логов. 0 - отключено // По умолчанию: "0" af_debug_level "0" // Квар гарантирует работу плагина даже если Вы забыли поменять квар 'mp_forcecamera' или если его изменил другой плагин. // По сути является аналогом 'mp_forcecamera' // По умолчанию: "2" af_forcecamera "2" // Флаги доступа для игроков, которые смогут наблюдать за всеми, несмотря на ограничения связанные с кварами 'mp_forcecamera' и 'mp_forcechasecam' // По умолчанию: "d" af_access_flags "d" // Отображать (1) или нет (0) в Free Look меню все найденные камеры карты // По умолчанию: "1" af_use_trigger_camera "1" // Включить (1) или нет (0) в Free Look меню возможность смотреть за кем наблюдает мертвый игрок // По умолчанию: "1" af_look_for_dead "1" // Список команд (разделенных ';') с помощью которых будет вызываться 'Меню наблюдения' // 'impulse 100' - фонарь, 'impulse 201' - спрей // если используется say, то say_team добавляется автоматически // По умолчанию: "say /look" af_menu_cmd "say /look" // Запоминать (1) или нет (0) выбранного в Free Look меню игрока, за которым велось наблюдение (если он умер) и // восстанавливать наблюдение за ним, когда он возродится // По умолчанию: "1" af_lock_target "1"
