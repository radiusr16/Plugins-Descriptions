Warmup 1.3.0

Автор: Radius

Описание:

Плагин позволяет организовать на сервере разминку в начале каждой карты. Обладает широкими возможностями настройки.

Особенности :

  • поддержка плагина Health Grenade by Radius (см описание команды warmup_add_items)
  • поддержка плагина Ultimate Weapons by SKAJIbnEJIb (см описание команды warmup_add_items)
  • поддержка CDSM мода (уже есть в архиве)
  • возможность сделать классическую разминку с выдачей денег и закупкой через стандартное меню
  • возможность сделать разминку на гранатах (см описание команды warmup_add_items)
  • возможность настройки списка оружия для каждой карты или префикса карт (команда warmup_add_items)
  • возможность добавления mp3 или wav звуков, которые будут проигрываться после начала разминки (команда precache_sound)
  • возможность выполнения команд из файлов перед стартом разминки и после завершения, в том числе с возможностью отключения определенных плагинов
  • после завершения разминки плагин отключается полностью (до начала следующей карты)
  • все квары и команды для удобства настраиваются в автоматически создаваемом конфигурационном файле 'addons/amxmodx/configs/warmup.cfg'
  • автоматически создаваемый словарь многоязычности 'addons/amxmodx/data/lang/warmup.txt'

Настройки (из автоматически создаваемого конфигурационного файла 'warmup.cfg'):

// Уровень дебаг-логов. 0 - отключено // По умолчанию: "0" warmup_debug_level "0" // Продолжительность разминки в секундах // По умолчанию: "45" warmup_duration_time "45" // Блокировать (1) или нет (0) возможность выбрасывать оружие во время разминки // По умолчанию: "0" warmup_block_drop_weapon "0" // Количество единиц здоровья при каждом возрождении на время разминки // По умолчанию: "100" warmup_health_limit "100" // Количество единиц брони при каждом возрождении на время разминки // По умолчанию: "100" warmup_armor_limit "100" // Количество денег, которое будет выдаваться при каждом возрождении во время разминки // По умолчанию: "16000" warmup_money "16000" // Файл с командами, которые необходимо выполнить перед стартом разминки // По умолчанию: "warmup_start.cfg" warmup_start_cmd_file "warmup_start.cfg" // Файл с командами, которые необходимо выполнить после завершения разминки // По умолчанию: "warmup_end.cfg" warmup_end_cmd_file "warmup_end.cfg" // RGB цвет и XY координаты HUD сообщений плагина // По умолчанию: "139 139 0 | -1.0 0.2" warmup_hud_msg "139 139 0 | -1.0 0.2" // Номер звука (начиная с 1), который будет проигрываться во время разминки // Значение 0 - отключает функцию, -1 - случайный номер // По умолчанию: "-1" warmup_sound_number "-1" // Список задержек между рестартами в секундах. Количество задержек определяет также количество рестартов // По умолчанию: "3, 1, 1" warmup_restarts_list "3, 1, 1" // Формат команды: precache_sound "путь к файлу звука" // формат звука: mp3 или wav // папку sound/ можно не указывать как для wav, так и для mp3 precache_sound "radio/com_go.wav" // Формат команды: warmup_add_items "идентификатор(ы) оружия из списка" // идентификатор(ы) оружия - обязательный аргумент, отвечающий за список оружия на время разминки // Через пробел можно указывать количество боезапаса. Например: warmup_add_items "hegren 255" - разминка на гранатах // Если боезапас не указан, выдается максимально возможный для покупки // Идентификаторы, если их несколько, разделяются ';' (точка с запятой) // Внимание: первым должно быть указано основное оружие на котором проводится разминка // Список доступного оружия: p228, scout, xm1014, mac10, aug, elite, // fiveseven, ump45, sg550, galil, famas, usp, glock18, awp, mp5navy, // m249, m3, m4a1, tmp, g3sg1, deagle, sg552, ak47, knife, p90 // knife, hegren, sgren, flash // Например: warmup_add_items "ak47;knife;hegren" warmup_add_items "knife" warmup_add_items "usp" warmup_add_items "glock18" warmup_add_items "deagle" warmup_add_items "m4a1" warmup_add_items "awp" warmup_add_items "ak47" warmup_add_items "famas" warmup_add_items "aug" warmup_add_items "scout" warmup_add_items "galil" warmup_add_items "sg552" warmup_add_items "mac10" warmup_add_items "mp5navy" warmup_add_items "tmp" warmup_add_items "ump45" warmup_add_items "p90" warmup_add_items "m249"
